## Dataset

The dataset contains:

- 10,000 synthetic patient healthcare records
- Demographic variables
- Smoking indicators (SNOMED coded)
- Family history of asthma
- Blood pressure measurements
- Cholesterol measurements
- Environmental exposure indicators
- Asthma deterioration target labels


## Project Workflow

### 1. Business Understanding
- NHS asthma deterioration prediction
- Population health management
- Proactive clinical intervention

### 2. Data Understanding
- NULL analysis
- Outlier detection
- SNOMED validation
- Distribution analysis

### 3. Data Preparation
- SQL cleaning pipelines
- Clinical feature engineering
- CTE-based modelling views
- Data quality validation

### 4. Modelling
- Exploratory Data Analysis (EDA)
- Train/test partitioning
- Classification modelling
- ROC/AUC evaluation

### 5. Deployment Recommendation
- Architecture comparison
- NHS-scale analytics considerations
- Kappa architecture proposal

## Key Engineered Features

The project includes several engineered healthcare features such as:

- Smoking risk indicators
- Family asthma history flags
- Blood pressure trend metrics
- Cholesterol delta calculations
- Comorbidity index
- Motorway pollution risk bands
- Hypertension indicators
- Pulse pressure calculations
