# Healthcare Analytics for Asthma Risk Prediction

An end-to-end healthcare analytics pipeline developed using SQL and MATLAB to predict worsening asthma symptoms using synthetic NHS-style patient data.

This project applies the CRISP-DM methodology to perform data engineering, feature engineering, predictive modelling, and deployment architecture design for large-scale healthcare analytics.

---

## Project Overview

Asthma affects millions of people across the UK and places significant pressure on NHS resources. Early identification of patients at risk of worsening asthma symptoms can support proactive healthcare interventions and reduce emergency admissions.

This project was developed as part of a Big Data and Data Mining healthcare analytics case study, focusing on:

- SQL-based clinical data preparation
- Feature engineering using healthcare indicators
- Machine learning classification models
- Predictive analytics for NHS-style systems
- Scalable deployment architecture recommendations

---

**Objectives**

- Analyse synthetic healthcare records
- Clean and validate clinical data using SQL
- Engineer predictive healthcare features
- Train and compare machine learning models
- Evaluate predictive performance using clinical metrics
- Recommend scalable NHS deployment architecture

---

**Tech Stack**:
- SQL, CTE pipelines
- MATLAB
- Logistic Regression, Decision Tree, Random Forest (TreeBagger)

**Methodology**: CRISP-DM Framework

**Domain**: 
- Healthcare Analytics
- Predictive Modelling
